Flare occurrence as a function of McIntosh spot class
Data combined over 1969 - 1976 (Kildahl 1980; M- and X-class events)
and over December 1988 - June 1996 (NOAA/SWPC assocations of C-, M-, 
and X-class flares with active regions)
   [see Bloomfield et al. (2012), ApJ, 747, L41 for details on their
    application in Poisson forecasting and full verification testing]
CLS = McIntosh active region class code
C = # C-class flares observed from that spot class
      [decimal because Kildahl (1980) does not report C-class numbers
       so NOAA/SWPC C-class rates are used over both periods]
M = # M-class flares observed from that spot class
X = # X-class flares observed from that spot class
N = # separate daily occurrences of that spot class
